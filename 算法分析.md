 # 算法
 ## 一：时间复杂度 T(n)代表执行的次数
 ### 1. 
 - T(n)=O(f(n))
 - 随着问题规模n的增大，算法执行时间的增长率和f（n）的增长率相同

### 2. 程序运行的总时间主要和两点有关
- 执行每条语句的耗时
- 每条语句的执行频率（主要）

### 3. 计算频度
```c
for(int i=1;i<=n;i++)     //外层，频度为 n+1
{
    for(int j=1;j<=n;j++)  //外层n，自己n+1，频度为 n*（n+1）
    {
        c[i][j] = 0;       //外层n，内层n，频度为 n*n

        for(int k=1;k<=n;k++)//外层n，内层n，自己n+1，频度 n*n*（n+1）
        {
            c[i][j] = c[i][j] + a[i][k]*b[k][j]; //频度n*n*n
        }
    }
}
```
- f(n)=n+1+n*(n+1)+n*n+n*n*(n+1)+n*n*n

      =2n*n*n+3n*n+2n+1 

### 4. 计算时间复杂度
   - 可以忽略所有低次幂和最高次幂的系数，可以简化算法分析
        f(n)=2n*n*n+3n*n+2n+1
        T(n) = O(n³) 

### 5. 
- 最好时间复杂度：最好情况
- 最坏时间复杂度：最坏情况
- 平均时间复杂度：平均值
### 通常只讨论最坏情况的时间复杂度


## 二：计算时间复杂度
### 1. - 常量阶 
例：
```c
for(int i=0;i<1000;i++)
{
    x++;   //频度为1
    s = 0; //频度为1
}

f(n)=1+1=2
T(n)=O(1)
```

### 2. - 线性阶
例：
```c
for(int i = 0 ; i < n ; i++)  //频度n+1
{
    x++;   //频度为n
    s = 0; //频度为n
}

f(n)=n+n+（n+1）=3n+1
T(n)=O(n)
```

### 3. - 平方阶
例：
```c
x = 0;   //频度为1
y = 0;   //频度为1
for(int k=1;k<=n;k++)  //频度为n+1
{
    x++;    //频度为n
}

for(int i=1;i<=n;i++)  //频度n+1
{
    for(int j=1;j<=n;j++)  //频度n*(n+1)
    {
        y++;   //频度n*n
    }
}

T(n)=O(n²)
```

### 4. - 立方阶
例：
```c
x=1;    //频度为1
for(int i=1;i<=n;i++)  //频度为n
{
    for(int j=1; j<=i ;j++)  //频度为（1+2+3+···+n）=n（n+1）/2
    {
        for(int k=1;k<=j;k++) //频度为 1+（1+2）+（1+2+3)+···+(1+2+3+···n)=[n(n+1)(n+2)]/6
        {
            x++;
        }
    }
}

T(n)=O(n³)
```

### 5. - 对数阶
例：
```c
for(int i=1;i<=n;i=i*2)
{
    x++;
    s = 0;
}
```
